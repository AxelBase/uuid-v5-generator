Privacy Policy

Privacy Policy

Effective Date: November 29, 2025

Welcome to the UUID v5 (Namespace-Based) Generator by AxelBase. This tool runs 100% in your browser. We are deeply committed to your privacy — so much so that we built this utility with zero data collection by design.

We Collect Nothing

This tool does not collect, store, transmit, or process any personal data. Specifically:

  • No IP addresses are logged
  • No browser fingerprinting or device information
  • No input data (namespaces, names, generated UUIDs) ever leaves your device
  • No cookies, localStorage persistence, or session tracking
  • No analytics, pixels, or third-party scripts

All UUID generation happens using the native Web Crypto API (crypto.subtle.digest('SHA-1', ...)) entirely on your machine. Even if you generate 10,000 IDs, nothing is sent anywhere.

How the Tool Works

Every feature — single generation, bulk download, copy-to-clipboard — is executed client-side only. The site is a Progressive Web App (PWA) that works fully offline after the first visit. No backend. No database. No logs.

Third-Party Services

We use only Bootstrap 5.3.3 via jsDelivr CDN for styling. This CDN does not track users or set cookies when used statically. No other external services are loaded.

Your Data Security

Since no data is transmitted, the strongest security guarantee possible is provided: if it never leaves your device, it can’t be intercepted. You control everything.

Children’s Privacy

This tool is not directed at children under 13. We do not knowingly collect any personal information from children.

Changes to This Policy

We may update this policy if new features are added that change data practices. Any changes will be reflected here with an updated effective date.

Contact

Questions? Open an issue on our GitHub repository.
